Project description

The purpose of this project is to develop a cost-effective solution for toxic gases detection by using wireless sensor networks. The solution consists of five sensor nodes and a gateway router connected to cloud platform. Machine learning and Artificial intelligent modules will be utilized to analyze the collected gases and predict other gases that could be exist with the collected gasses. GPS module will be used to locate the leaked gas and the coverage of the sensor network. Routing energy aware will be taken into consideration while developing the solution.
